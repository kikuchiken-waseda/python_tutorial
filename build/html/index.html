注釈

このページに関して

プログラミング初学者用に作成しているので， 種々イントロダクションや，環境設定に関しては必ずしも最良とは限りません.

  • つまり，比較的初心者でも使用可能だったり理解可能な内容を記述します．
    • そのため, 一部説明には正確性が欠けていたり，より良いツールがあったりします．
    • その点，御留意ください．
  • これらの情報は 2019 年度の最新の情報になります．
  • また Mac OS を前提をした記述です
    • 基本的には最新版の OS (Mojave) を前提としますので， そうでない場合には事前にアップデートをしておいてください
    • https://support.apple.com/ja-jp/macos/mojave

いくつかの項目は 事前準備必須 となっていることに注意してください．
